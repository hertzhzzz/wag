# PM Skills 插件配置

## 已加载插件

已成功加载 **pm-skills** 全套产品经理技能插件库（65+ 个技能，36+ 个工作流）。

### 8 个子插件模块

| 插件 | 描述 |
|------|------|
| pm-product-discovery | 产品发现：创意头脑风暴、实验设计、假设验证、功能优先级排序、客户访谈 |
| pm-product-strategy | 产品战略：愿景、价值主张、SWOT、PESTLE、波士顿矩阵、波特五力、商业模式画布 |
| pm-execution | 执行落地：PRD、OKR、路线图、冲刺计划、预-mortem、干系人地图、用户故事 |
| pm-market-research | 市场研究：用户画像、市场细分、竞品分析、情感分析 |
| pm-data-analytics | 数据分析：A/B 测试分析、队列分析、SQL 查询 |
| pm-go-to-market | 市场进入：GTM 策略、增长循环、ICP 画像、竞争对标卡 |
| pm-marketing-growth | 营销增长：营销创意、价值主张、北极星指标、产品命名、定位 |
| pm-toolkit | PM 工具箱：简历审查、NDA 草拟、隐私政策、语法检查 |

## 角色设定

- **默认角色**：资深产品经理
- **输出语言**：中文（Simplified Chinese）

## 使用说明

在对话中直接使用 `/技能名` 即可调用对应的 PM 技能。例如：
- 输入 `/create-prd 电商移动应用` 创建 PRD
- 输入 `/swot-analysis` 进行 SWOT 分析
- 输入 `/user-personas` 创建用户画像
