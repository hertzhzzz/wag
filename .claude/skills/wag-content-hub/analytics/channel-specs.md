# WAG Channel Specifications

Technical requirements and platform rules for each content channel.

## LinkedIn

| Spec | Value |
|------|-------|
| **Hook limit** | 210 characters (hard) |
| **First 90 chars** | Must make sense standalone (LinkedIn shows ~3 lines before "see more") |
| **Post length** | 150-300 words |
| **Hashtag limit** | 6-10 |
| **CTA format** | Experience-based question (not generic) |
| **Link policy** | NO external links in post body (-60% reach penalty). Put links in first comment. |
| **Best posting time** | Tue-Thu, 9-11 AM AEDT |
| **Algorithm notes** | Personal profile > Company page by 561% reach; carousel posts get 6.6% avg ER |

### LinkedIn Content Formats (by ER)

| Format | ER Benchmark | Best For |
|--------|-------------|----------|
| Document Carousel | 6.6% | Process explanations |
| 3-Step Framework | 33% (WAG data) | Verification how-tos |
| Story + Framework | 10-15% | Case studies |
| Text-only | 4% | Brand intros |
| Brand intro | 9% | Awareness |

### LinkedIn Character Counter
- Hook: 210 chars max
- Body: ~2000 chars max recommended
- CTA: 100 chars max
- First comment: 1000 chars max

## X (Twitter)

| Spec | Value |
|------|-------|
| **Hook limit** | 280 characters (hard) |
| **Post length** | 100-280 chars standalone |
| **Hashtag limit** | 1-2 |
| **CTA format** | Poll > Retweet > Comment |
| **Link policy** | Links in reply or second tweet |
| **Best posting time** | Tue-Thu, 9-11 AM AEDT |
| **Algorithm notes** | Contrarian hooks get more engagement; threads perform well for complex topics |

### X Hook Formulas (Ranked)

| Rank | Formula | Example |
|------|---------|---------|
| 1 | "Wrong/Here's the thing" | "Wrong. Most Australian brands verify factories wrong." |
| 2 | "One fact [audience] doesn't know" | "One fact most China buyers don't know..." |
| 3 | "Hot take:" | "Hot take: Alibaba is killing Aussie small business." |
| 4 | Question-based | "What's your factory verification process?" |

### X Content Types

| Type | Characters | Best For |
|------|------------|----------|
| Single tweet | 100-280 | Sharp insights |
| 2-tweet thread | 50-100 + 150-200 | Contrarian + proof |
| Poll | 50-100 + options | Engagement testing |

## Facebook

| Spec | Value |
|------|-------|
| **Hook limit** | 150 chars for link preview |
| **Post length** | Flexible, 100-500 words |
| **Hashtag limit** | 3-5 |
| **CTA priority** | Saves > Shares > Comments |
| **Best posting time** | 7-9 PM AEST (evening) |
| **Algorithm notes** | Community relationship content; saves = long-term value |

### Facebook Hook Types

| Type | Formula | Example |
|------|---------|---------|
| Relationship | "We've all been there..." | "You know that moment when..." |
| Story | "A client paid $X and..." | "A client flew to China and found..." |
| List | "3 things to check before..." | "3 signs your supplier is legit:" |
| Question | "Has this happened to you?" | "Has a supplier ever ghosted you?" |

### Facebook Save-Worthy Format
- Educational listicle format performs best for saves
- Practical, immediately applicable content
- "Save this for later" implicit value

## SEO/Blog

| Spec | Value |
|------|-------|
| **Meta description** | 160 characters max |
| **Post length** | 1500-3000 words |
| **Hashtag limit** | 5-10 (in body) |
| **CTA type** | Internal links > External links |
| **Best posting time** | Publish Mon-Wed for indexing |
| **Algorithm notes** | Long-form guides perform for long-tail discovery |

### SEO Blog Structure

| Section | Purpose | Length |
|---------|---------|--------|
| Hook | Problem statement | 1-2 paragraphs |
| Introduction | Why this matters | 2-3 paragraphs |
| Body | Step-by-step or topic sections | 1500-2000 words |
| Conclusion | Summary + CTA | 1-2 paragraphs |
| Resources | Internal/external links | As needed |

### Blog CTA Types

| CTA Type | Purpose | Placement |
|----------|---------|-----------|
| Internal link | Keep reader on site | In body |
| External link | Cite sources | In resources section |
| Newsletter | Build email list | End of post |
| Contact | Drive inquiry | End of post |
