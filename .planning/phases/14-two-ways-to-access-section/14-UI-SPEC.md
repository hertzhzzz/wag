## Animation Specification

**Type:** IntersectionObserver scroll-triggered fade-in-up

**Implementation (from HowItWorks.tsx lines 44-59):**
```typescript
const [visible, setVisible] = useState(false)
const sectionRef = useRef<HTMLDivElement>(null)

useEffect(() => {
  const observer = new IntersectionObserver(
    (entries) => {
      if (entries[0].isIntersecting) {
        setVisible(true)
      }
    },
    { threshold: 0.15 }
  )

  if (sectionRef.current) {
    observer.observe(sectionRef.current)
  }

  return () => observer.disconnect()
}, [])
```

**Card animation:**
```typescript
className={`transition-all duration-700 ${
  visible ? 'opacity-100 translate-y-0' : 'opacity-0 translate-y-8'
}`}
style={{ transitionDelay: `${idx * 150}ms` }}
```

**Timing values:**
| Property | Value |
|----------|-------|
| Duration | 700ms |
| Stagger delay | 150ms between cards (idx * 150) |
| Transform | `translate-y-8` → `translate-y-0` |
| Opacity | 0 → 1 |
| Threshold | 0.15 (15% of section visible) |

**Easing:** Default CSS transition (ease-out behavior)

**Reduced motion accessibility:**
```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
  .opacity-0.translate-y-8 {
    opacity: 1;
    transform: translate-y-0;
  }
}
```

**Reference notes from animate skill:**
- 150ms stagger is optimal for 2 cards (not too fast, not too slow)
- 700ms duration for entrance is appropriate (not laggy)
- prefers-reduced-motion must be supported

## AI Slop Detection

**Reference notes from critique skill:**

**Anti-Patterns Verdict:** PASS — No AI slop detected.

**Analysis:**
- Two side-by-side cards is a standard, professional B2B pattern — not an AI fingerprint
- Compass and Database icons are contextually appropriate (navigation/exploration vs data access)
- Chinese bullet points add authenticity rather than appearing as generic placeholder text
- Amber emphasis is restrained and professional, not the purple-blue gradient AI slop
- Section header "How Would You Like to Find Your Factory?" is conversational and human, not generic marketing copy
- No glassmorphism, no gradient text, no dark mode with glowing accents
- IBM Plex Serif headings are a distinctive brand choice, not generic Inter/Roboto

**What's Working:**
1. Equal visual weight (50/50) for both options respects user autonomy
2. Amber emphasis on Full Service guides attention without being pushy
3. Chinese bullet points signal authentic China expertise
4. Professional typography hierarchy using IBM Plex

**Priority Issues:** None — design is clean and professional.
