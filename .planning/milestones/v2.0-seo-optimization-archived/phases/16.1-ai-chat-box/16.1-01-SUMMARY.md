### 2. Key Technical Changes

**State Machine Flow (Before → After):**
```
Before: idle → greeting → options → selected → (2s timeout) → form → submitted
After:  idle → greeting → options → selected → (user clicks "Get Details") → form → submitted
```

**Error Handling (Before → After):**
```
Before: Inline `<p className="text-red-500">` below form fields
After:  Toast banner at top of chat content, auto-dismisses after 4s
```

**Icon Change:**
```
Before: MessageCircle (generic chat)
After:  Bot (explicit AI association)
```

**Mobile Responsiveness:**
```
Before: w-80 → w-96 (fixed widths)
After:  w-[80vw] sm:w-96 max-w-[340px] (fluid with constraints)
```
