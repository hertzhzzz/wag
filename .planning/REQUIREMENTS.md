# Requirements: WAG Website

**Defined:** 2026-03-17
**Core Value:** Improve mobile responsive layout to ensure the website provides an excellent user experience on all devices

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Deployment

- [ ] **DEPLOY-01**: Website is deployed to Vercel production environment
- [ ] **DEPLOY-02**: Custom domain winningadventure.com.au is configured and accessible
- [ ] **DEPLOY-03**: SSL certificate is issued and HTTPS works
- [ ] **DEPLOY-04**: Environment variables (Supabase, Resend) are configured in Vercel

### Mobile Navigation

- [ ] **MOBILE-01**: Navbar remains visible when scrolling down on mobile devices
- [ ] **MOBILE-02**: Navbar is clickable/tappable when scrolled down on mobile

### Social Links

- [ ] **SOCIAL-01**: Facebook link is added to Footer alongside existing LinkedIn
