# WAG SEO Ranking Project

## What This Is

Winning Adventure Global (WAG) SEO project — improve Google Australia rankings for target keywords to drive enquiry form submissions (discovery calls). WAG is a service business: they connect Australian businesses with verified Chinese factories via in-person factory tours. NOT a product seller. Core differentiator: trust and verification for businesses afraid of being scammed on 1688/Alibaba.

## Core Value

Drive qualified enquiry form submissions from Australian businesses searching for Chinese factory verification and inspection services.
