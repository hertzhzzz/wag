# WAG Website

Winning Adventure Global official website.

[![Security Scan](https://github.com/hertzhzzz/wag/actions/workflows/security.yml/badge.svg)](https://github.com/hertzhzzz/wag/actions/workflows/security.yml)
[![Deploy to Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/hertzhzzz/wag)

## Tech Stack

- Next.js 14.2 (App Router)
- TypeScript
- Tailwind CSS
- Supabase (Auth + DB)
- Resend (Email)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Deployment

Deploy to Vercel by pushing to the master branch.
