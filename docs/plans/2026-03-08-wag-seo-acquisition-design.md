---

## 三、技术 SEO 基础

### 3.1 URL 结构
- 现有 `/resources/[slug]` 结构适合英文博客
- 确保每篇文章有清晰的英文 slug

### 3.2 Meta 信息
- 每个页面配置独立的 title、description
- 使用 Next.js Metadata API

### 3.3 站点地图
- 提交 Google Search Console
- 生成 XML sitemap

### 3.4 内链策略
- 文章之间相互链接
- 形成内容集群（Cluster）
