## 1.3 贸易商身份确认

### 1.3.1 经营范围性质判定

**基于现有信息无法完成官方判定**，但通过多维度证据交叉分析，可形成高度可能的推断：

**证据一：注册地址物理属性**。主体A提供的地址"北京国际眼镜大厦"为北京地区知名的眼镜专业批发市场，主要功能为**商贸展示与批发零售**，而非工业生产。该建筑不具备制造所需的厂房设施、环保审批及安全生产条件 。

**证据二：海关数据贸易模式**。NBD平台记录显示，主体A的出口产品以眼镜架（HS 90031900）为主，辅以眼镜布、眼镜托盘等配件，呈现**"核心产品+配套周边"的组合模式**，符合贸易公司的典型操作——通过产品组合提升订单附加值，降低单一产品依赖风险 。出口产品明确标注"VERATI""HANADI"等品牌，表明其**贴牌代工或品牌授权贸易**属性，而非自有品牌制造 。

**证据三：进出口结构不对称**。进口记录仅2条、出口记录22条的**严重不对称结构**，符合**纯出口型贸易公司**特征。制造企业通常呈现原材料进口-成品出口的双向流动，而贸易公司则主要体现为出口单向流动 。

**【推断】** 主体A的实际经营模式高度可能为：以北京国际眼镜大厦为办公据点，通过eyewearglobo.com进行海外客户开发，接单后委托国内实际生产厂家生产，自身并不具备生产制造能力。网站刻意使用"Factory""Manufacturer"等表述，存在**误导客户的虚假宣传嫌疑** 。
