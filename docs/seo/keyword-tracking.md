## How to Check Keyword Rankings

### Method 1: Google Search Console (Recommended)

1. Go to Google Search Console
2. Select your property (winningadventure.com.au)
3. Navigate to Performance → Pages
4. Search for your target keywords in the search bar
5. Record the average position shown

### Method 2: Manual Search (For Accurate Australian Results)

1. Open incognito/private browsing window
2. Set Google domain to `.com.au` (or use Google Australia: https://www.google.com.au)
3. Search for your target keyword
4. Find where your page appears in results
5. Record the position number

**Tip:** Bookmark this search URL for quick weekly checks:
- `https://www.google.com.au/search?q=epic+sourcing+winningadventure`
